## Локальная установка

Для удобной работы локально можно создать симлинки из корня репозитория:
```bash
mkdir -p .cursor/agents
mkdir -p .cursor/skills

ln -s ~/Projects/ai-bro/.cursor/agents/developer.md ./.cursor/agents/
ln -s ~/Projects/ai-bro/.cursor/agents/reviewer.md ./.cursor/agents/

ln -s ~/Projects/ai-bro/.cursor/skills/bro-do-it ./.cursor/skills/
ln -s ~/Projects/ai-bro/.cursor/skills/bro-give-me-plan ./.cursor/skills/
ln -s ~/Projects/ai-bro/.cursor/skills/bro-debug ./.cursor/skills/
ln -s ~/Projects/ai-bro/.cursor/skills/bro-reflect-about-us ./.cursor/skills/
ln -s ~/Projects/ai-bro/.cursor/skills/bro-think-with-me ./.cursor/skills/
```
