# Формат вывода bro-remember-it

Оркестратор передаёт пользователю ответ thinker (можно слегка сжать без потери смысла).

Перед списком рекомендаций **всегда**:

```markdown
## Memory-bank

Путь: `<подтверждённый корень>`
```

Далее — разделы, минимальный каркас рекомендации:

```markdown
#### Рекомендация 1
- **действие**: дополнить
- **куда**: `memory-bank/features/...`
- **что**: ...
- **почему**: ...
```

Финальный блок оркестратора:

```markdown
## Что обязательно зафиксировать

- рекомендации: 1, 3 | ничего
- почему: ...

Запись в файлы — только после вашего подтверждения. Напишите, какие номера принимаете, или уточните target.
```
