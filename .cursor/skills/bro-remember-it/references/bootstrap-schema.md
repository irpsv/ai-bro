# Bootstrap

Используй только когда пользователь выбрал вариант «memory-bank ещё нет» или подтвердил создание структуры позже. Thinker может **рекомендовать** пути по этой схеме; **не** создавай файлы автоматически.

| Тип продуктового знания | Куда |
| --- | --- |
| Устройство системы, инварианты, контракты, интеграции | `memory-bank/architecture/...` |
| Фича: сценарии, роли, ограничения, edge cases | `memory-bank/features/...` |
| Воспроизводимые типовые действия | `memory-bank/playbooks/...` |
