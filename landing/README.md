# Landing для GitHub Pages

## Структура

```text
landing/
├── index.html
└── styles.css
```

## Как опубликовать на GitHub Pages

1. Откройте настройки репозитория на GitHub.
2. Перейдите в раздел **Pages**.
3. В `Build and deployment` выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/landing`
4. Сохраните настройки и дождитесь публикации.
