# Init memory bank — первичная инициализация

Первичное создание банка знаний, когда `memory-bank/index.md` ещё не существует.

## Правила workflow

1. **ОБЯЗАТЕЛЬНО** соблюдай [структуру memory bank](../references/memory-bank-structure.md) и [правила содержимого](../references/content-rules.md).
2. Если `memory-bank/index.md` уже есть — **ОСТАНОВИСЬ** и предложи `/bro-remember-it`. Не перезаписывай банк.

## ПОРЯДОК РАБОТЫ

1. **ПРОВЕРЬ**, что `memory-bank/index.md` отсутствует. Если файл есть — **ОСТАНОВИСЬ** и предложи `/bro-remember-it`.
2. **ОПРЕДЕЛИ** путь к банку (по умолчанию `memory-bank/` в корне репозитория). **СПРОСИ** подтверждение у человека.
3. **ОСТАНОВИСЬ** и дождись ответа.
4. **ЗАПУСТИ** скрипт инициализации:
   ```bash
   bash <путь-к-скиллу>/scripts/init-memory-bank.sh <путь-к-банку>
   ```
   Путь к скиллу — каталог, откуда загружен `bro-init-memory-bank` (например `skills/bro-init-memory-bank/` в репозитории ai-bro или аналог после установки).
5. **СОБЕРИ** базовый контекст проекта из репозитория: README, package-файлы, структура каталогов, CI-конфиги.
6. **ЗАПОЛНИ** ключевые файлы в `project/`:
   - `overview.md`
   - `tech-stack.md`
   - `architecture.md`
   Заполняй на основе собранного контекста. Неизвестное оставь с пометкой для уточнения.
7. **СПРОСИ** у человека, что ещё важно зафиксировать сразу: фичи, playbooks, соглашения, глоссарий. Один вопрос за раз.
8. **СОЗДАЙ** запрошенные файлы в `features/` и `playbooks/` по [правилам содержимого](../references/content-rules.md). Обнови соответствующие `index.md`.
9. **ПРОВЕРЬ** корневой `index.md`: ссылки только на `project/*.md`, `features/index.md`, `playbooks/index.md`.
10. **ПОКАЖИ** человеку итог: что создано, что заполнено, что требует уточнения. Для дальнейшего пополнения укажи `/bro-remember-it`.

## Критерии завершения

- Структура каталогов соответствует схеме memory bank.
- `project/overview.md`, `project/tech-stack.md`, `project/architecture.md` содержат осмысленный черновик или подтверждённые факты.
- Человек видит карту банка и знает, что дальше знания добавляет `/bro-remember-it`.
