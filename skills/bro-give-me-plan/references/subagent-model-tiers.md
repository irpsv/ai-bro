# subagent-model-tiers

Единый словарь тиров и семейств моделей.
Этот файл описывает не slug-и конкретного harness, а семейства в формате `vendor/model-family`. Сначала выбери тир по типу работы, затем семейство модели внутри этого тира.

## Общие правила

1. Для каждого запуска субагента сначала определи тир по типу работы, затем выбери семейство модели внутри этого тира.
2. Семейства моделей внутри каждого тира перечислены в порядке приоритета. Всегда выбирай первое доступное семейство из списка выбранного тира.
3. Глобальный приоритет вендоров: `cursor` -> `openai` -> `anthropic` -> `google`.
4. Если более приоритетное семейство недоступно в текущем harness, переходи к следующему семейству по списку в рамках того же тира.
5. Не повышай тир, если нет доступной модели. Если ни одно семейство выбранного тира недоступно, используй либо **auto** режим, либо текущую **inherit** модель.
6. После выбора тира выбери целевую глубину размышлений (`reasoning` / `thinking`) для этого тира.
7. Канонические уровни глубины размышлений в этом файле: `low`, `balanced`, `deep`.
8. Глубина размышлений зависит от модели. Используй только те параметры, варианты или отдельные model id, которые реально поддерживает выбранное семейство в текущем harness.
9. `low` означает минимальную глубину размышлений, если она поддерживается моделью.
10. `balanced` означает `medium`, если такой уровень есть у модели, иначе обычный режим без принудительного `high`.
11. `deep` означает максимальную доступную глубину размышлений, например `high`, thinking variant или другой усиленный режим модели.
12. Если модель не поддерживает настройку глубины размышлений, используй ее обычный режим и не меняй тир из-за отсутствия такой настройки.

## analytics

Используй только для действительно сложной аналитики:

- root cause investigation;
- архитектурный разбор;
- критика плана;
- synthesis по нескольким источникам с неоднозначностью;
- trade-off analysis при конфликтующих ограничениях.

Целевая глубина размышлений: `deep`.

Подходящие семейства моделей:

1. `openai/gpt`
2. `anthropic/claude-opus`
