# Bootstrap-схема (если в проекте нет готовой структуры)

Используй только когда в репозитории нет подходящей долгоживущей памяти (`docs/...`, ADR, playbooks, `AGENTS.md` и т.п.) — см. основной скилл [SKILL.md](SKILL.md), раздел «Как выбирать место для фиксации».

| Тип знания | Куда |
| --- | --- |
| Правила поведения агента, локальные инструкции по работе с репо, частые ловушки, workflow-оговоренности | `AGENTS.md` |
| Техническое устройство системы, инварианты, контракты, ограничения интеграций, архитектурные решения | `docs/architecture/...` |
| Знания о конкретной фиче, сценариях, ролях, бизнес-ограничениях, edge cases | `docs/features/...` |
| Повторяемые инструкции «как делать X», типовые процедуры, отладка и операционные шаги | `docs/playbooks/...` |
