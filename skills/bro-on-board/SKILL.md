# Bro On Board

Твоя задача — познакомить человека с набором скиллов ai-bro и при необходимости **инициализировать memory bank** проекта.

**НЕ ПОПОЛНЯЙ** уже существующий банк знаний — для этого есть скилл пополнения (`/bro-remember-it`).
**НЕ СОЗДАВАЙ** артефакты планирования в `plans/` — для планов есть скилл планирования (`/bro-give-me-plan`).

## ОБЩИЕ ПРАВИЛА

1. **ОБЯЗАТЕЛЬНО** соблюдай [правила диалога](./references/dialog-rules.md)!
2. **ОБЯЗАТЕЛЬНО** соблюдай и **НЕ нарушай** [порядок работы](#порядок-работы)!
3. Сначала самостоятельно собери контекст: есть ли уже memory bank, что за проект.
4. Не спрашивай то, что можно установить с высокой уверенностью.
5. Держи онбординг коротким: карта скиллов и следующий шаг, без полного разбора всех workflow.

## ОГРАНИЧЕНИЯ / STOP

1. **ЗАПРЕЩЕНО** перезаписывать существующий memory bank.
2. **ЗАПРЕЩЕНО** создавать или изменять файлы в `plans/`.
3. **ЗАПРЕЩЕНО** пополнять банк сверх первичного заполнения при init — дальнейшие знания фиксирует `/bro-remember-it`.
4. **ЗАПРЕЩЕНО** придумывать факты о проекте.
5. **ЗАПРЕЩЕНО** читать оба workflow подряд без необходимости. Сначала определи режим, потом читай только нужный.

## ПОРЯДОК РАБОТЫ

1. **ОПРЕДЕЛИ** режим по [правилам выбора workflow](#выбор-workflow).
2. **ПРОЧИТАЙ** только выбранный workflow и **СОБЛЮДАЙ** его инструкции.
3. После завершения **ПОКАЖИ** человеку итог и один понятный следующий шаг.

## Выбор workflow

**ЗАПРЕЩЕНО** читать оба workflow подряд.

- Если человек просит только инициализацию memory bank — выбери [init-memory-bank](./workflows/init-memory-bank.md).
- Во всех остальных случаях (знакомство, «с чего начать», первый заход) — выбери [tour](./workflows/tour.md).

Если банк уже есть, а человек просит init — **ОСТАНОВИСЬ**, сообщи об этом и **ПРЕДЛОЖИ** `/bro-remember-it` для пополнения.

## Quality Control

Перед завершением проверь:

- человек понимает, какой скилл когда вызывать;
- если банк создавался — структура соответствует схеме, `plans/` не заполнен планами;
- если банк уже был — он не перезаписан;
- итог заканчивается одним конкретным следующим шагом.
