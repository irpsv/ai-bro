# Карта скиллов ai-bro

Краткий справочник для онбординга. Не подменяй им чтение workflow других скиллов.

| Этап | Команда | Зачем |
|---|---|---|
| Думаем | `/bro-think-with-me` | Прояснить мысль, проверить идею, выбрать подход |
| Дебажим | `/bro-debug` | Найти причину бага по фактам, без сразу фикса |
| Планируем | `/bro-give-me-plan` | Составить однозначный план реализации |
| Делаем | `/bro-do-it` | Реализовать по плану или чёткой задаче + ревью |
| Вайб-код | `/bro-vibe-code` | Сразу к реализации без отдельного плана |
| Инициализация банка | `/bro-init-memory-bank` | Создать memory bank с нуля |
| Помним | `/bro-remember-it` | Пополнить существующий memory bank |
| Онбординг | `/bro-on-board` | Знакомство с набором скиллов |

## Типовой флоу

1. Прояснить задачу — `/bro-think-with-me` (или `/bro-debug`, если это баг).
2. Зафиксировать план — `/bro-give-me-plan` (или сразу `/bro-vibe-code`, если план не нужен).
3. Реализовать — `/bro-do-it`.
4. Запомнить устойчивые знания — `/bro-remember-it`.

## Memory bank

Постоянный слой знаний проекта: `project/`, `features/`, `playbooks/`.  
Раздел `plans/` — временные артефакты планирования, их создаёт скилл планирования.

- Банка ещё нет → инициализация через `/bro-init-memory-bank`.
- Банк уже есть → пополнение через `/bro-remember-it`.
