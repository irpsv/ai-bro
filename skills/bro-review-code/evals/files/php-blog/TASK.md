# Задача

Добавить к блогу публичные комментарии к постам и модерацию автором поста.

## Критерии приемки

- авторизованный пользователь может оставить комментарий к опубликованному посту;
- автор поста может удалить любой комментарий под своим постом;
- автор комментария может удалить только свой комментарий;
- HTML в тексте комментария не исполняется в браузере;
- список комментариев открывается вместе с постом, без лишних запросов на каждый комментарий;
- неавторизованный пользователь не может создавать или удалять комментарии;
- форма комментария защищена от CSRF;
- есть тесты на права удаления и на экранирование HTML.

## Объект ревью

Все файлы в `evals/files/php-blog/` — изменения фичи комментариев относительно простого блога без комментариев.
